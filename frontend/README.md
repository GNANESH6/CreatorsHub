# CreatorsHub - Frontend

This is the React-based frontend for the CreatorsHub platform.

## 🛠️ Built With

- **React 19**
- **Vite**
- **React Router 7**
- **Socket.io-client**
- **Axios**
- **Lucide React**

## 📖 Main Documentation

For full project details including the tech stack, features, and setup instructions, please refer to the [Root README](../README.md).

## 🚀 Running Locally

```bash
npm install
npm run dev
```
